# Case Study 2: Time Series Forecasting with XGBoost

## Project Overview
This project models monthly truck sales data using ensemble tree-based methods, with a focus on **XGBoost** for time series forecasting.

---

## Dataset
The data is given from Kaggle
https://www.kaggle.com/datasets/ddosad/dummy-truck-sales-for-time-series/data
